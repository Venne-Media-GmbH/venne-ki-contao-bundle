# Venne KI — Contao Bundle

[![CI](https://github.com/Venne-Media-GmbH/venne-ki-contao-bundle/actions/workflows/ci.yml/badge.svg)](https://github.com/Venne-Media-GmbH/venne-ki-contao-bundle/actions/workflows/ci.yml)
[![License](https://img.shields.io/packagist/l/venne-media/venne-ki-contao-bundle.svg)](LICENSE)

Bindet das [Venne KI](https://venne-ki.de) Chat-Widget in deine Contao-5-Seite ein.

## Installation

```bash
composer require venne-media/venne-ki-contao-bundle
```

Oder im Contao Manager: `venne-media/venne-ki-contao-bundle` hinzufügen.

## Einrichtung

1. Bot im [Venne KI-Portal](https://venne-ki.de) anlegen.
2. Im Contao-Backend → **System → Venne KI** die Bot-UUID einfügen.
3. Speichern.

Das Widget erscheint ab sofort auf jeder Frontend-Seite.

## Anforderungen

- Contao 5.3+
- PHP 8.2+
