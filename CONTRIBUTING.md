# Contributing

Pull Requests sind willkommen.

## Setup

```bash
git clone git@github.com:Venne-Media-GmbH/venne-ki-contao-bundle.git
cd venne-ki-contao-bundle
composer install
```

## Vor dem PR

```bash
composer qa
```

Läuft PHP-CS-Fixer (Dry-Run), PHPStan Level 8 und PHPUnit.

## Lizenz

Beiträge laufen unter LGPL-3.0-or-later.
